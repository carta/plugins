# Smart Documents

Generate and send legal documents (LP Agreements, Investment Management Agreements, formation docs) via the Carta CLI.

## Skills

| Skill | Description |
|-------|-------------|
| `generate-smart-document` | > |
| `send-smart-document-signature` | > |

## Prerequisites

- Carta CLI installed and authenticated (`carta login`)
- Write scope enabled (`carta scope set write`)
- Access to a fund with smart document templates configured

## Usage

```
Generate an LP Agreement for my fund
```

```
Send smart document 118 for signature
```
